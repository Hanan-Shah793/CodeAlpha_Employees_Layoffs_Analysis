# CodeAlpha_Employees_Layoffs_Dataset

# 📊 Employee Layoffs Analysis (EDA & Dashboard)

## 📌 Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship**.

The objective of this project is to analyze global employee layoff data using Exploratory Data Analysis (EDA), data visualization, and dashboarding techniques to identify trends, patterns, and business insights.

## 🎯 Objectives

- Clean and preprocess the dataset
- Handle missing values and duplicates
- Perform Exploratory Data Analysis (EDA)
- Analyze layoffs by company, industry, location, stage, and country
- Visualize trends using charts and graphs
- Build an interactive dashboard
- Extract meaningful business insights

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI 

## 📊 Analysis Performed

- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Company-wise Layoff Analysis
- Industry-wise Layoff Analysis
- Country-wise Layoff Analysis
- Location-wise Analysis
- Stage-wise Analysis
- Funding vs Layoffs Analysis
- Percentage Layoff Analysis
- Dashboard Creation

## 📈 Visualizations

The project includes:

- Horizontal Bar Charts
- Scatter Plots
- Trend Analysis
- Top 10 Companies Analysis
- Top 10 Industries Analysis
- Top Countries Analysis
- Funding vs Layoffs Comparison

## 🔍 Key Insights

- Amazon, Intel, and Meta reported some of the highest layoffs.
- Retail, Consumer, and Other industries experienced significant workforce reductions.
- The SF Bay Area, Seattle, and New York City were among the most affected locations.
- The United States recorded the highest number of layoffs.
- Funding does not always correlate with lower layoffs.
